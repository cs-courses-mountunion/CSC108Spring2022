 Assignment:
Write a Processing program that draws three circles in a 400 x 400 drawing window that have different letters drawn on them.  The program should respond to mouse clicks on the circles by displaying a word on screen that begins with the same letter as the one in the circle that was clicked upon.  The program should also respond to key presses on letters displayed in the circles by displaying the same word on screen that begins with the letter typed.

void drawCircles() {
  stroke(255);
  fill(0);
  textSize(84);

  ellipse(110,  100, 80, 80);
  ellipse(210,  100, 80, 80);
  ellipse(310,  100, 80, 80);

  fill(255);
  text(firstLetter,  82, 130);
  text(secondLetter, 182, 130);
  text(thirdLetter, 282, 130);

  stroke(204); 
} // end of drawCircles
Your Task:
First of all, change the program so that it displays your initials in the circles.  If you don't have a middle name or if your initials contain duplicate letters, choose other letters so that three different letters are displayed in the circles.  You must have three different letters displayed by your program.

You must add code to the draw function of this program to respond to user input in the following manner.
  • When the user types a key on the keyboard,
    Set the background color of the window to 204, and call the drawCircles function.   Then use an if statement to test the key that was typed — if that is either the upper-case version of the letter in the left circle, or the lower-case version of that letter, then change the fill color to some color you like and use the text function to display a word that starts with the letter in the left circle at screen position (10, 370).

    Use more if statements to respond to key presses that correspond to the letters shown in the other two circles.  Use different colors and be sure that each word displayed starts with the letters shown in the circles.

    See Example 5-18 in the "Type" section of Chapter 5 in the textbook to see how to test for both the upper- and lower-case version of a key.
  • When the user clicks the mouse,
    Set the background color of the window to 204, and call the drawCircles function.   Then use an if statement to test the distance of the mouse position from the center of the left-most circle — if that distance is less than half the size of the circle, then change the fill color to some color you like and use the text function to display the same word that starts with the letter in the left circle as was used in the keyboard processing described above.  Display the word at screen position (10, 370).

    Use more if statements test to see if the mouse click occurred inside the center or the right circle, and display the same word that was used in the keyboard processing for that letter.  Use the same color for each word, no matter if it is displayed because of a mouse event or a keyboard event.

    See Example 5-14 in the "Location" section of Chapter 5 in the textbook to see how to test for a mouse position being less than a certain distance from a point.
