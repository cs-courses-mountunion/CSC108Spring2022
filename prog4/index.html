Assignment:
Write a Processing program that "connects the dots" when the user clicks the mouse in various locations on in the output window.  If the user presses the space bar, the screen should be reset to blank.  Each mouse click causes the program to "remember" the location of the mouse when the click occurred.
Sample:
Your program should work just like this sample.  Right-click on the link in the preceding sentence and save the file in your CSC108 folder.  Then using File Explorer, right-click on the .zip file and choose "Extract All" from the pop-up menu.  After extracting the files from the .zip file, there will be a file with a .exe extension – double-click on the file to run the demo and see how your program should work.

Run the demo program, move the mouse around the screen, then click either mouse button several times while moving the mouse around, then press the spacebar on the keyboard, then click either mouse button again several times.

Your Task, Step 1:  Declare and Instantiate one int variable and two int Arrays
Above the setup function in your program, declare an int variable named numClicks and set it to zero.  Then declare two Arrays that can store int values.  One of the arrays will keep track of the horizontal positions of the mouse clicks, and the other array will keep track of the vertical positions of the mouse clicks.  It would make sense to call these arrays x and y, but you can use any names you like.  These arrays should be set up initially to store 500 values each.  Refer to Example 11-9 in the textbook for an example of how to do this.
Your Task, Step 2:  Write the draw function for the program
Here is an outline of the logic you should use for the draw function of your program, in order to get it to work as the sample does:
  1. Set the background to a medium to dark color that you like
  2. if (the spacebar has been pressed):
    1. store the value zero in numClicks
  3. if (the mouse has been clicked):
    1. add 1 to numClicks
    2. store mouseX in x[numClicks]
    3. store mouseY in y[numClicks]
  4. set the stroke color to some color you like
  5. set the fill color to some color you like
  6. for each value of i from 1 up to and including numClicks:
    1. draw an ellipse centered at the point (x[i], y[i]) that is 30 × 30 pixels in size
    2. draw a line from (x[i], y[i]) to (x[i-1], y[i-1])
  7. set the stroke color to some color you like that is different than the one used above
  8. set the fill color to some color you like that is different than the one used above
  9. draw an ellipse centered at the point (mouseX, mouseY) that is 30 × 30 pixels in size
  10. draw a line from (mouseX, mouseY) to (x[numClicks], y[numClicks])
