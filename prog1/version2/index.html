CSC 108    —     Program # 1    —     Due:  1/21/22

Assignment:
Write a Processing program that draws a white circle at the mouse position in a drawing window that is sized 700 x 500 pixels, and draws a red circle that chases the white circle around the screen, eventually catching the white circle if the mouse movement is stopped.
Sample:
Your program should work just like this sample.  Right-click on the link in the preceding sentence and save the file in your CSC108 folder.  Then using File Explorer, right-click on the .zip file and choose "Extract All" from the pop-up menu.  After extracting the files from the .zip file, there will be a file with a .exe extension – double-click on the file to run the demo and see how your program should work.
Starting Point:
To get started on this assignment, make a new Processing sketch, then copy Example 2-2 from Chapter 2 of the text into the editor.

Example 2-2:


Change the parameters in the size function so that the screen size is correct for this problem.  Next, delete all of the lines inside the draw function EXCEPT for the last one (the "ellipse(mouseX, mouseY, 80, 80);" line).  If you run the program now, you should see a bunch of circles being drawn over one another.

Paste the following code as the first lines that are inside the curly braces of your draw function: 
        // clear screen and set background color
        background(204);

        // set filling color for circle at mouse position
        fill(255);
If you run the program now, the trail of circles won't be there; only one circle should be drawn, with its center at the mouse position.

Next, paste this line at the very top of your program (before the "void setup" line): 
        float x, y;
Then paste these lines inside the curly braces of the setup function: 
        x = 350;
        y = 250;
Finally, at the end of your draw function, set the fill color to a red shade (use the Processing menu option Tools → Color Selector as described on pp. 28 and 29 of the textbook to pick a shade of red that you like), and then draw an ellipse that is centered at (x, y) and is 80 pixels wide and high.

If you run the program now, you should have a red circle in the center of the screen, with a white circle following your mouse around.  Ask your instructor for help if you have trouble getting to this point.
Your Task:
Right now, your program always draws a red circle at the same position in the window.  You need to add code to the draw function of your program that will calculate a new position for the red circle's center based on the old position of its center and where the mouse is located in the window.  To do this, you should add a fraction of the distance from the mouse location to the center of the red circle to both the x and the y variables (the fraction is needed to make the circle "chase" the mouse -- if we added the entire distance, then the circles would always be on top of one another).

We need to make two calculations, one for the horizontal and one for the vertical.  Consider the horizontal position of the center of the red circle:  the distance between the centers of the circles can be computed by subtracting the horizontal position of the red center from the mouse's horizontal position.  This value should be divided by 30, and then that value should be added to the horizontal position of the red circle's center to set a new value for the horizontal position of the center of the red circle.  You can do this either before or after the red circle is drawn in the draw function; either way will work.  Do a similar thing for the vertical position of the center of the red circle using the vertical position of the mouse.

Extras:
Just for fun, feel free to add some extras to your program.  For example, can you change the plain white circle at the mouse position into a Pumpkin or an Easter Egg or something else?  Have fun experimenting with Processing functions and statements.

If you would like, add this code to the very bottom of your draw function: 
        if (mousePressed) {
            x = y = 700;
        }
This will add the ability to reposition the red circle off the screen whenever the mouse button is clicked.  Give it a try!
